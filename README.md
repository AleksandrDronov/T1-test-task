# Тестовое задание

### Запуск проекта:

1. Клонировать репозиторий.

   git clone git@github.com:AleksandrDronov/T1-test-task.git

2. Войти в папку проекта.

   cd T1-test-task

3. Установить зависимости.

   npm install

4. Запустить Json server.

   npm run server

5. Запустить проект в браузере.
